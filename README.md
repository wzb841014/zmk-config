# ZMK Config for Corne

My ZMK keyboard configuration for Corne (CRKBD) with nice_nano_v2.

## Features
- Bluetooth (BLE)
- SSD1306 Display
- RGB Underglow
- Battery Reporting
- NKRO

## Build
Push to main branch triggers GitHub Actions build.

## Download Firmware
1. Go to Actions tab
2. Select latest workflow run
3. Download zmk-firmware.uf2
